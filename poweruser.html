AUTOMATE TEDIOUS TASKS WITH OUR EXCEL ADD-IN UTILITIES

COPY-PASTE VISIBLE CELLS ONLY

A typical nightmare in Excel is when you want to copy filtered data from a column to another column. Excel will just ignore the filter and potentially ruin your data set! Wit Power-user you can simply copy and paste visible cells only to get yourself out of this painful situation.


UNPIVOT TABLE

This tool can save you hours. A situation that happens in Excel is when you have a crosstab table, but you want the data in the form of a database, where each row is duplicated for every column of your initial table. That’s essential to create PivotTables for instance. Now that’s something you can do instantly with Power-user’s UnPivot Table!


MERGED CELLS MANAGER

Merged cells can be a nightmare for many Excel users, especially in databases or if you make VBA macros. Click the Merged Cells Manager and you will get a list of all merged cells in the active sheet. From there you will be able to view each merge cells, to unmerge them or to unmerge and fill them (a dream tool so you can create PivotTables).


ADVANCED FUNCTIONS

Get new powerful functions to use in your Excel formulas. These functions include SumColor, CountColor, CountUnique, CountVisible, IsMerged, HasFormulas, UsedRange, LastCell, Vlookup_Max, CAGR, and dozens more, but also Stock functions that will get market and financial data directly in your spreadsheet in near-real time.


DELETE EMPTY CELLS

Sometimes when working with data on Microsoft Excel, you need to move cells up eveytime there is a blank value. That is just what this utility will do for you, allowing you to remove empty cells from any selection.


CLEAN FORMULAS

hen Excel formulas become complex, it does not help people understand or correct them if there are unnecessary redundancies. If you are working on Sheet1, you don’t need to refer to ‘Sheet1!A1’ but just to ‘A1’. Press Clean Formulas and Power-user will clean all formulas from such redundancies.

CHANGE FORMULAS ABSOLUTE / RELATIVE

If you realize after a while that you have used a relative ‘A1’ reference in many cells while you should have used an absolute ‘A1’… it’s not too late. This Power-user feature will let you make conversions in formulas between absolute, relative, relative column and absolute row or absolute column and relative row.
